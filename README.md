## 🚀 使用说明

### 阅读 Markdown 笔记

Obsidian 格式的 `.md` 笔记（主要在 `01-02-03` 课程中）可以直接：

1. 用 [Obsidian](https://obsidian.md/) 打开本仓库根目录作为 vault，体验最佳；
2. 或用任意 Markdown 编辑器/文本编辑器查看。

> ⚠️ 部分笔记中可能存在 `[[双链]]` 指向本仓库未包含的其他学科（如线性代数），在新仓库中可能失效，但不影响主体内容阅读。

### 下载 PDF 资料

PDF 复习资料统一放在各课程目录下的 `资料/` 文件夹中。

由于 PDF 文件较大，本仓库使用 **Git LFS** 管理 `.pdf` 文件。直接点击 GitHub 网页上的 PDF 可以在线预览或下载；若要完整 clone 到本地，请先安装 Git LFS：

```bash
# macOS
brew install git-lfs

# 安装后在本仓库执行
git lfs install
git lfs pull
```

- **笔记**：为个人精读整理，按章节组织，含公式推导、核心理解与易错点。
